Code Golf

Minimum characters. Maximum impact. Precision in every byte.

About the Event

Code Golf is a competitive programming challenge where the goal isn't just to solve the problem, but to do so using the fewest number of characters possible. It's a battle of linguistic wizardry, algorithmic shortcuts, and esoteric language features.

Participants are given a series of logical challenges and must submit their solutions in languages like Python, JavaScript, C++, or Ruby. The leaderboard is ranked based on byte count – the shorter the code, the higher the score.

Tournament Format

  • Round 1 — 5 logic puzzles to be solved in under 60 minutes.
  • Round 2 — High-complexity algorithms with a focus on extreme minification.
  • Final Decider — One ultimate problem where only the shortest solution across all languages wins.

Rules & Guidelines

  • Individual participation only.
  • Supported Languages: Python 3, JavaScript (Node.js), C++17, Ruby 3.0.
  • Score = Total characters in the source code (ignoring whitespaces in some rounds).
  • No external libraries or modules allowed unless specified.
  • Standard Input/Output must be used for reading data and printing results.
  • Plagiarism detection enabled; original code is mandatory.
  • Judging criteria: Correctness first, then Byte Count.
